# 论文批量读取系统 - 前端

React 19 + TypeScript + Vite + Tailwind CSS 4.x 前端项目。

## 技术栈

- **React 19** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS 4.x** - 样式框架
- **React Router DOM** - 路由管理
- **Axios** - HTTP 客户端
- **pnpm** - 包管理器

## 安装

```bash
cd playground
pnpm install
```

## 开发

```bash
pnpm dev
```

访问 http://localhost:5173

## 构建

```bash
pnpm build
```

## 项目结构

```
src/
├── api/          # API 客户端
├── components/   # React 组件
├── pages/        # 页面组件
├── hooks/        # 自定义 Hooks
└── utils/        # 工具函数
```
